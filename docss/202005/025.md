# Docker 设置容器时间

> 镜像内运行一下命令

```script
ln -snf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
```

> 使用命令 `date` 查看时间是否正确

++**修改之后必须要重启其他连接工具（如`Java`程序等）**++

---

临时

> 进入 `docker` 容器

```
docker exec -it [容器ID] /bin/bash
```

> 执行以下命令

```shell
export TZ=Asia/Shanghai
```

> 使用命令 `date` 查看时间是否正确
